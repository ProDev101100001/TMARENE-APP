# **App Name**: Tamareeni

## Core Features:

- User Authentication: Secure user authentication via Google, Facebook, Apple, email/password, and phone number (OTP).
- Initial Onboarding: Collects user's physical information, goals, and preferences to tailor fitness plans.
- Step Counter: Tracks daily steps, distance, and calories burned using device sensors. Includes a customizable daily goal.
- 30-Day Workout Program: Generates a personalized 30-day workout plan based on onboarding data, including exercise details, videos, and progress tracking.
- AI Calorie Estimation: Estimates calories in meals by analyzing photos using GPT-4o Vision, providing ingredient lists, portion sizes, and macro breakdowns. The system then saves a picture of the food in the user's food log.
- AI Health Assistant: An AI-powered chatbot tool that analyzes user habits, provides personalized health advice, suggests healthier alternatives, and answers nutrition/exercise questions.
- Weight Tracking: Tracks and displays expected weight changes based on calorie intake and expenditure, with weekly weight update requests and plan adjustments.

## Style Guidelines:

- Primary color: Emerald green (#50C878), evokes health and vitality.
- Background color: Very light desaturated green (#E0F8E8), creating a calm and supportive environment.
- Accent color: Soft teal (#70DBD1), provides a complementary and inviting highlight.
- Body and headline font: 'PT Sans', a humanist sans-serif to combine a modern look and a little warmth or personality.
- Use modern and friendly icons that visually represent health and fitness activities.
- Clean and intuitive layout, prioritizing user-friendliness and easy navigation.
- Subtle, motivational animations to celebrate milestones and encourage engagement.
